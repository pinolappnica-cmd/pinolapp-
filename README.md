# PinolApp Nica Delivery

Plataforma completa de delivery de comida construida con **Next.js (App Router)**, **TypeScript**, **Tailwind CSS**, **Prisma ORM** y **PostgreSQL**.

Incluye:
- Frontend para clientes (registro, login, restaurantes, menú, carrito, órdenes, perfil).
- Panel de administración (gestión de restaurantes, menús, órdenes y repartidores).
- API REST con rutas para autenticación, restaurantes, menú, carrito, órdenes, admin y drivers.
- Base de datos con Prisma y PostgreSQL.
- Middleware para proteger rutas según rol (admin, cliente, driver).
- Integración con Google Maps para seguimiento de entregas.
